# Booking-App

Monorepo containing Frontend (Vite + React) and Backend (Django).

Structure:

- `Frontend/Booking-App` — React + Vite frontend
- `Backend` — Django backend and env

Instructions

1. Frontend
   - Install: `npm install` inside `Frontend/Booking-App`
   - Start: `npm run dev`

2. Backend
   - Create virtualenv and install dependencies from `Backend/packages.txt`
   - Run migrations and start the Django server
